A simple web application for organizing and joining pickup soccer games.

## Features

- **Create Games**: Schedule pickup soccer games with location, date, time, and player limits
- **Join Games**: Browse and join upcoming games
- **Persistent Storage**: Games are saved locally in your browser
- **Responsive Design**: Works on desktop and mobile devices

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/Pickup-soccer.git
cd Pickup-soccer
```

2. Open `index.html` in your web browser

That's it! No build process or dependencies required.

## Usage

1. **Create a Game**: Fill out the form with game details and click "Create Game"
2. **Join a Game**: Browse upcoming games and click "Join Game" to participate
3. **Track Players**: See how many players have joined each game

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage for data persistence
